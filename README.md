# 🛍️ Customer Shopping Behavior Analysis

An end-to-end data analytics project that analyzes customer shopping behavior using **Python, MySQL, SQL, and Power BI**. The project focuses on data cleaning, exploratory data analysis (EDA), SQL-based business insights, and interactive dashboard creation to help understand customer purchasing patterns.

---

## 📌 Project Overview

This project demonstrates a complete data analytics workflow:

* Importing and exploring customer shopping data
* Cleaning and preprocessing the dataset using Python
* Loading data into MySQL using SQLAlchemy
* Performing business analysis using SQL queries
* Creating an interactive Power BI dashboard for visualization

## 🚀 How to Run the Project

1. Download or clone the repository.
2. Install the required Python libraries.
3. Open the Jupyter Notebook.
4. Connect Python to your MySQL database.
5. Load the dataset into MySQL.
6. Execute the SQL queries.
7. Open the Power BI dashboard and refresh the data.
